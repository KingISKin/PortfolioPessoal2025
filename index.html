Sobre Mim

Sou desenvolvedor fullstack com 5 anos de experiência, atuando no desenvolvimento de soluções web completas. No front-end, trabalho com HTML, CSS e JavaScript (ES6+), utilizando frameworks como React para criar interfaces modernas e responsivas. No back-end, desenvolvo APIs RESTful robustas utilizando Node.js e Express, além de trabalhar com bancos de dados relacionais como PostgreSQL e MySQL, e também NoSQL como MongoDB.

Também sou familiarizado com práticas essenciais de DevOps, como controle de versão com Git, containerização com Docker e deploy em plataformas cloud. Busco sempre aplicar boas práticas de desenvolvimento, testes automatizados e arquitetura limpa para entregar soluções escaláveis e de alta qualidade, alinhadas às necessidades do negócio!
