Sobre Mim

Meu nome é João Victor, tenho 22 anos e sou desenvolvedor fullstack com 5 anos de experiência no mercado. Tenho domínio em tecnologias front-end como HTML, CSS e JavaScript (ES6+), utilizando frameworks modernos como React para criar interfaces responsivas e eficientes. No back-end, trabalho com Node.js e Express, construindo APIs RESTful robustas e seguras, além de experiência com bancos de dados relacionais (PostgreSQL, MySQL) e NoSQL (MongoDB).

Também sou familiarizado com práticas essenciais de DevOps, como controle de versão com Git, containerização com Docker e deploy em plataformas cloud. Busco sempre aplicar boas práticas de desenvolvimento, testes automatizados e arquitetura limpa para entregar soluções escaláveis e de alta qualidade, alinhadas às necessidades do negócio!
